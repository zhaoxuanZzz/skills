# 提示词助手 Skill (Prompt Assistant Skill)

帮助用户生成、优化和调整高质量的AI提示词。

## 功能特性

### 🎯 核心功能

1. **提示词生成** - 为不同场景生成定制化提示词
2. **提示词优化** - 改进现有提示词的清晰度和有效性
3. **场景匹配** - 根据使用场景推荐合适的提示词模板
4. **Markdown输出** - 所有提示词都以格式化markdown输出

### 📋 支持的场景

- **数据解析** - 结构化数据提取，JSON/CSV输出
- **复杂任务** - 多步骤分解，逐步指导
- **Markdown文档** - 格式化输出，支持章节、表格、图表
- **角色扮演** - 定义AI的专业角色和个性
- **系统提示词** - 创建系统级的AI行为定义

### 📚 支持资源

#### 模板库
- `templates/` 目录包含：
  - 数据解析模板
  - 复杂任务分解模板
  - Markdown结构化输出模板
  - 提示词优化指南
  - 角色系统提示词模板

#### 参考资料
- `reference/` 目录包含：
  - 最佳实践 - 编写高效提示词的原则
  - 常见模式 - 12种经过验证的提示词框架
  - 反模式 - 应避免的常见错误
  - 工具特定提示词 - 针对不同AI工具的优化

## 快速开始

### 使用场景 1: 数据解析提示词

```
用户: "我需要从文本发票中提取供应商和金额"

助手的做法:
1. 询问输入格式和数据量
2. 确认期望的输出格式
3. 推荐 "数据解析模板"
4. 生成带有JSON schema和示例的提示词
```

### 使用场景 2: 优化现有提示词

```
用户: "我的提示词给出了不一致的结果"

助手的做法:
1. 分析提示词的问题（不清楚、缺乏示例等）
2. 提出改进方案
3. 重建优化的提示词
4. 解释改进的部分
```

### 使用场景 3: 生成Markdown文档提示词

```
用户: "我需要生成API文档"

助手的做法:
1. 理解API的性质
2. 推荐 "Markdown结构化输出模板"
3. 生成包含：
   - 标题层级指导
   - 格式化规则
   - 示例结构
   - 质量检查清单
```

## 核心模板说明

### 1. 数据解析模板
**用途**: 从非结构化数据中提取结构化信息
**输出**: 包含schema、验证规则和错误处理的提示词
**常见场景**: 发票解析、简历提取、日志分析

### 2. 复杂任务分解模板
**用途**: 将大型、多步骤的任务分解成可管理的阶段
**输出**: 包含阶段、步骤、验证标准的详细计划
**常见场景**: 内容创建、分析报告、项目规划

### 3. Markdown结构化输出模板
**用途**: 生成格式化的、有明确结构的markdown文件
**输出**: 包含格式规则、视觉元素、内容指南的提示词
**常见场景**: 文档、教程、报告

### 4. 提示词优化指南
**用途**: 改进现有提示词
**方法**: 分析、识别问题、提出改进、创建优化版本
**工具**: 包含检查清单、问题映射表、案例研究

### 5. 角色系统提示词模板
**用途**: 定义AI的角色、个性、能力范围
**输出**: 系统级提示词，用于整个对话会话
**例子**: 产品经理助手、技术写作者、创意策划者

## 最佳实践快速查看

### ✅ 做
- 清晰的目标陈述
- 具体的约束条件
- 真实的示例
- 多种输出格式
- 验证标准

### ❌ 不做
- 模糊的需求
- 假设AI的知识
- 单个无代表性的示例
- 相冲突的指令
- 缺少成功标准

## 工作流程

### 标准工作流程

1. **理解用户需求**
   - 提出问题理解：任务目标、输入格式、输出格式
   - 确定场景分类

2. **选择合适模板**
   - 数据解析 - 用于结构化提取
   - 复杂分解 - 用于多步骤任务
   - Markdown - 用于文档生成
   - 优化 - 用于改进现有提示词
   - 系统提示词 - 用于定义AI行为

3. **生成定制提示词**
   - 使用模板作为基础
   - 填充用户特定的细节
   - 添加示例和约束
   - 确保清晰和完整

4. **展示为Markdown**
   - 使用markdown代码块
   - 清晰的部分标记
   - 说明如何使用

5. **提供优化建议**
   - 询问是否需要调整
   - 建议常见定制
   - 提供测试指导

## 常见问题

### Q: 这个skill能直接执行任务吗？
**A:** 不能。这个skill生成提示词，你可以将其复制到其他AI工具中使用。

### Q: 生成的提示词能立即使用吗？
**A:** 大多数情况下可以，但最好针对你的特定情况进行定制。

### Q: 如何知道提示词是否有效？
**A:** 参考最佳实践部分的质量检查清单。

### Q: 支持哪些语言？
**A:** 提示词可以生成任何你要求的语言。默认是中文和英文。

### Q: 可以为特定工具（ChatGPT、Claude等）优化吗？
**A:** 可以。参考 `tools-specific-prompts.md` 了解针对特定工具的优化。
