# Web Presentation Skill

A skill for creating modern, interactive web presentations as single-file HTML artifacts.

## Quick Start

**For Users:**
1. Read `SKILL.md` for complete workflow and guidelines
2. Open `templates/viewer.html` as your starting point
3. Customize the `PRESENTATION` config and `SLIDES` array
4. Open the HTML file in any browser

**For Reference:**
- `reference/layouts.md` - All available slide layouts
- `reference/themes.md` - Color themes and customization
- `examples/product-launch-demo.html` - Complete working example

## What This Skill Does

Creates professional web presentations with:
- 6 different slide layouts (title, content, two-column, image+text, quote, section)
- Smooth keyboard and touch navigation
- Customizable themes and colors
- Progress tracking and fullscreen mode
- Fully responsive design
- Single self-contained HTML file output

## Key Features

- **No build process** - Edit HTML and refresh
- **Keyboard shortcuts** - Arrow keys, space, F for fullscreen
- **Mobile friendly** - Swipe gestures supported
- **Accessible** - WCAG AA compliant colors
- **Offline capable** - Only fonts require internet
- **Claude artifacts** - Can be displayed directly in claude.ai
